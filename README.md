dotfiles
========

Use `ln` and `git` to manage linux dot files.

Usage of `ln`
-------------

```shell
ln -s /path/to/dotfiles/.vimrc ~/.vimrc
ln -s `pwd`/.vimrc ~/.vimrc
```

Install
-------

    $ git clone --recursive https://github.com/yixinglu/dotfiles.git
    $ cd dotfiles
    $ ./install.sh

macOS
-----

* ripgrep
